SHACL Validation on Object Store¶

Ocean Info Hub SHACL validation on S3(minio) objects

It should be noted here that SHACL validation is not a service OIH offers. Rather, the validation is a capacity that the OIH architectural approach facilities. Further this validation follows W3C recommendations as describted in https://www.w3.org/TR/shacl/.

Flow¶

  • get an object (use the dask notebook)

  • process the object against OIH SHACL shapes

def label_status (row):
  result = row['http://www.w3.org/ns/shacl#resultSeverity']
  if result == "nan":
    return "NA" 
  elif "Warning" in result:
    return "Warning"
  elif "Violation" in result:
    return "Violation"  
  else:
    return result

def source_shape (row):
  result = row['http://www.w3.org/ns/shacl#sourceShape']
  if type(result) is list: 
    return result[0]['@id']
  else:
    return "NA"
